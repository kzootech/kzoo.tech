System Installation

The system should be installed using a USB Flash drive with the install67.fs image written to it.

Full disk encryption is recommended and should be setup by following the instructions in the Disk Setup section of the OpenBSD FAQ. Initialize the disk for UEFI booting when using fdisk(8). During installation with a USB drive, the encrypted deviced will be 'sd2'.

Starting sshd(8) by default is NOT recommended.

The X Window System should be configured to be started by xenodm(1).

A local user should be created.

Using the default auto-allocated layout is not recommended.

All filesets should be selected for installation.

System Configuration

Once an internet connection is established, syspatch(8) and fw_update(1) should be run to install all firmware and to apply all system patches.
