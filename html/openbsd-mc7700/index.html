Sierra Wireless MC7700 WWAN Modem

The Sierra Wireless MC7700 is a MiniPCI cellular modem card designed for use in laptops.

The Sierra Wireless MC7700 modem can come configured in either DirectIP Mode or QSM mode. This guides assumes the modem is configured in QSM mode.

The modem's mode can be switched by changing the effective PCI ID of the device. To set the modem to QSM mode change the PCI ID of the device to 68a2 as shown below.

x230# cu -l /dev/cuaU2
Connected to /dev/cuaU2 (speed 9600)
AT!ENTERCND="OEM password"
OK
AT!UDPID=68A2
OK
AT!RESET
OK

[EOT]
x230#

The device mode can be changed to DirectIP by changing the PCI ID to 68A3 in a similar fashion as above. Changing the device to 68A3 (DirectIP Mode) is NOT RECOMMENDED as this PCI ID is not authorized by the OEM X230 BIOS whitelist.
